# Overview

v-datepicker is a vue datepicker directive.

# Usage

```JavaScript
require('v-datepicker')
```

# Basic use:

```HTML
<input type="text" v-model="{modelName}"  v-datepicker="{modelName}" format="yyyy-MM-dd">
<input type="text" v-model="{modelName}"  v-datepicker="{modelName}" format="yyyy/MM/dd">
```
